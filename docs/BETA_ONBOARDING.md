## 📋 Scénarios à Tester

Nous vous encourageons à suivre ces 5 étapes clés pour découvrir l'outil :

1. **Créer votre premier dossier** de crédit (bouton "Nouveau Dossier").
2. **Saisir les bilans financiers** simplifiés et lancer le calcul automatique des ratios.
3. **Consulter la prédiction IA** pour comparer votre analyse avec le moteur de risque dématérialisé.
4. **Finaliser et Exporter** la note de synthèse au format **PDF** ou **Word**.
5. **Rechercher vos dossiers** passés sur le tableau de bord principal.

## 💡 À savoir durant cette phase

- **Données temporaires** : La base de données de staging est remise à zéro chaque semaine. Ne saisissez pas de données confidentielles réelles (anonymisez vos tests).
- **Fonctions absentes** : Pour cette beta, l'inscription libre et la réinitialisation de mot de passe ne sont pas encore actives.
- **Usage professionnel** : L'outil est optimisé pour une utilisation sur ordinateur (Desktop). Le support mobile n'est pas l'objectif de cette phase.
